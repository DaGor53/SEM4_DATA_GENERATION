# Курсовой проект.
# Разметка изображений.
Задача проекта состоит в разметке изображений таким образом, чтобы отметить на них наличие дронов. Полученная разметка может быть использована в задачах обучения моделей компьюетрного зрения на определения дронов на изображениях. 

Число изображений: 175 шт

Выводной формат: YOLO

Изображения: папка images

Метки: папка labels

Число классов: classes.txt (содержит только один класс - drones)

Заметки: notes.json

Разметка выполнена при помощи Label Studio. Установка и запуск выполнены в Docker командой:

docker run -it -p 8080:8080 -v %cd%\mydata:/label-studio/data heartexlabs/label-studio:latest

Label Studio запущена из браузера Chrome: http://localhost:8080

Конфигурация экспорта и процесс разметки представлены в прилагающихся .png файлах: "Процесс разметки.png", "Формат экспорта.png", "Экспорт.png"

# Инструкция для разметчика

1. Запустить Label Studio
2. Нажать Create в открывшемся меню
3. В разделе Project Name задать имя и описание проекта
4. В разделе Data Import загрузить батч изображений из папки на локальном компьютере
5. В разделе Labeling Setup выбрать Object Detection with Bounding Boxes, удалить преднастроенные Labels и добавить свою метку "drone".
6. Нажать Save
7. Выбрать первое изображение из открывшегося списка единоразовым щелчком левой клавиши мыши
8. Выбрать слева снизу класс drone и обозначить границы объекта. В случае наличия нескольких объектов данного класса на одном изображении допускается наложение границ объектов друг на друга
9. Нажать Submit в правом нижнем углу
10. Перейти к следующему изображению
11. Проделать операции 8-10 для всех изображений списка
12. В настройках экспорта использовать YOLO with Images для сохранения меток совместно с изображениями
